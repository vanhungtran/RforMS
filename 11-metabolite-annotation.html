11  Metabolite Annotation & Identification

Following spectral library search, this chapter covers the downstream annotation workflow: assigning chemical identities to detected features using MS1 mass accuracy, isotope patterns, adduct rules, and cross-database lookups. These steps transform a list of m/z values into a biologically interpretable metabolite inventory.
