# Mass Spectrometry Data Formats and Import

Understanding and working with various MS data formats is crucial for effective data analysis. This chapter covers the main data structures and file formats used in R for Mass Spectrometry.

## Overview of MS Data Formats

### mzML Format

The mzML format is the current standard for MS data exchange, defined by the Proteomics Standards Initiative (PSI).

```{r}
library(tidyverse)
library(Spectra)
library(msdata)
library(mzR)
library(ProtGenerics)

# Load example mzML file
mzml_file <- proteomics(full.names = TRUE, pattern = "TMT")
basename(mzml_file)
```

```{r}
# Read mzML data using Spectra
ms_data <- Spectra(mzml_file)
ms_data
```

### Understanding Spectra Objects

The `Spectra` object is the core data structure in R for Mass Spectrometry, providing an abstraction for MS data that allows efficient manipulation and analysis.

```{r}
# Basic information about the data
cat("Number of spectra:", length(ms_data), "\n")
cat("MS levels present:", paste(unique(msLevel(ms_data)), collapse = ", "), "\n")
cat("Retention time range:", range(rtime(ms_data)), "seconds\n")
```

```{r}
# MS level distribution
table(msLevel(ms_data))
```

### Core Spectra Variables

The `Spectra` class defines core variables that characterize each spectrum:

```{r}
# Core spectra variables
spectraVariables(ms_data) %>% head(15)
```

```{r}
# Access specific variables
# Retention times for first 10 spectra
rtime(ms_data)[1:10]
```

```{r}
# Precursor information for MS2 spectra
ms2_data <- filterMsLevel(ms_data, msLevel = 2L)
if (length(ms2_data) > 0) {
  precursor_info <- data.frame(
    scan = acquisitionNum(ms2_data)[1:10],
    precursor_mz = precursorMz(ms2_data)[1:10],
    precursor_charge = precursorCharge(ms2_data)[1:10],
    retention_time = rtime(ms2_data)[1:10]
  )
  print(precursor_info)
}
```

## Working with Spectra Objects

### Accessing Spectral Data

```{r}
# Get the first spectrum
first_spectrum <- ms_data[1]
mz(first_spectrum)[[1]] %>% head(10)
intensity(first_spectrum)[[1]] %>% head(10)
```

### Filtering Spectra

```{r}
# Filter by MS level
ms1_spectra <- filterMsLevel(ms_data, msLevel = 1)
ms2_spectra <- filterMsLevel(ms_data, msLevel = 2)

cat("MS1 spectra:", length(ms1_spectra), "\n")
cat("MS2 spectra:", length(ms2_spectra), "\n")
```

```{r}
# Filter by retention time
rt_filtered <- filterRt(ms_data, rt = c(1000, 2000))
length(rt_filtered)
```

### Metadata Extraction

```{r}
# Extract comprehensive metadata
spectra_df <- spectraData(ms_data) %>%
  as.data.frame() %>%
  select(msLevel, rtime, precursorMz, precursorCharge, 
         precursorIntensity) %>%
  head(10)

print(spectra_df)
```

## Quality Control and Data Inspection

### Basic QC Checks

```{r}
# Check for missing data
summary(is.na(precursorMz(ms_data)))
summary(is.na(rtime(ms_data)))
```

### Visualizing Data Overview

```{r}
library(ggplot2)

# Retention time distribution
rt_data <- data.frame(rtime = rtime(ms_data), msLevel = msLevel(ms_data))

ggplot(rt_data, aes(x = rtime, fill = factor(msLevel))) +
  geom_histogram(bins = 50, alpha = 0.7) +
  facet_wrap(~msLevel) +
  labs(title = "Retention Time Distribution by MS Level",
       x = "Retention Time (s)", y = "Count") +
  theme_minimal()
```
