# Introduction to Mass Spectrometry in R {.unnumbered}

This introduction provides hands-on examples of working with mass spectrometry data in R using the R for Mass Spectrometry ecosystem. We'll explore real datasets and demonstrate key functionalities.

## Loading Essential Packages

```{r}
# Load core MS packages
library(Spectra)           # Core MS data infrastructure
library(msdata)            # Example MS datasets
library(ProtGenerics)      # Generic functions for proteomics
library(PSMatch)           # Peptide-spectrum matching
library(tidyverse)         # Data manipulation
library(MsDataHub)         # Access to online MS resources

cat("Packages loaded successfully!\n")
```

## Exploring Example Datasets

The `msdata` package provides various example MS datasets for learning and testing.

### Proteomics Data

```{r}
# List available proteomics files
proteomics_files <- msdata::proteomics(full.names = TRUE)
cat("Available proteomics files:\n")
for (i in seq_along(proteomics_files)) {
  cat(sprintf("%d. %s\n", i, basename(proteomics_files[i])))
}

# Load a specific proteomics dataset (TMT experiment)
f <- msdata::proteomics(pattern = "2014", full.names = TRUE)
cat("\nSelected file:", basename(f), "\n")
```

### Loading and Examining MS Data

```{r}
# Create Spectra object
ms <- Spectra(f, backend = MsBackendMzR())

# Display basic information
cat("\nDataset summary:\n")
cat("  Total spectra:", length(ms), "\n")
cat("  MS levels:", paste(unique(msLevel(ms)), collapse = ", "), "\n")
cat("  RT range:", round(range(rtime(ms)), 2), "seconds\n")
cat("  m/z range:", round(range(unlist(mz(ms))), 2), "\n")

# Display the object
ms
```

## Peptide Fragment Calculation

The `PSMatch` package provides tools for theoretical peptide fragmentation.

```{r}
# Calculate theoretical fragments for a peptide
peptide_seq <- "THSQEEMQHMQR"

cat("Calculating theoretical fragments for:", peptide_seq, "\n\n")

fragments <- calculateFragments(peptide_seq)

# Display fragment information
cat("Theoretical fragments:\n")
print(head(fragments, 15))

cat("\nFragment types:", paste(unique(fragments$type), collapse = ", "), "\n")
cat("Total fragments:", nrow(fragments), "\n")
```

## Peptide-Spectrum Matching

Working with identification results from database searches.

```{r}
# Load identification files
id_files <- msdata::ident(full.names = TRUE)
cat("Available identification files:\n")
for (i in seq_along(id_files)) {
  cat(sprintf("%d. %s\n", i, basename(id_files[i])))
}

# Load PSM data
id <- PSM(id_files)

cat("\nPSM Summary:\n")
cat("  Total PSMs:", nrow(id), "\n")
cat("  Unique peptides:", length(unique(id$sequence)), "\n")
cat("  Unique proteins:", length(unique(id$DatabaseAccess)), "\n")

# Display first few PSMs
cat("\nFirst few PSMs:\n")
print(head(id, 5))
```

## Biological Annotation and Enrichment

Connecting MS data to biological databases for functional analysis.

```{r}
# Load Gene Ontology database
library(GO.db)

cat("Gene Ontology database loaded\n")
cat("Available columns:", paste(columns(GO.db), collapse = ", "), "\n")

# Query a specific GO term (focal adhesion)
go_term <- "GO:0005925"
go_info <- select(GO.db, 
                  keys = go_term,
                  columns = columns(GO.db),
                  keytype = "GOID")

cat("\nGO Term Information for", go_term, ":\n")
print(go_info)
```

```{r}
# Get human genes associated with focal adhesion
library(org.Hs.eg.db)

GO_0005925 <- AnnotationDbi::select(
  org.Hs.eg.db,
  keys = "GO:0005925",
  columns = c("ENTREZID", "SYMBOL"),
  keytype = "GO"
) %>%
  as_tibble() %>%
  filter(!duplicated(ENTREZID))

cat("\nGenes associated with Focal Adhesion (GO:0005925):\n")
cat("  Total genes:", nrow(GO_0005925), "\n")
print(head(GO_0005925, 10))
```

```{r}
# Get genes for another GO term (centrosome)
```
